# DeepLearn
### 李晓峰我来帮你了
1. 第一步  查看全部虚拟环境

   ```
   conda env list
   ```

2. 创建虚拟环境 ( 已有适合的环境就忽略 )

   ```
   conda create -n tf2101xxxx python=3.6
   ```

3. 切换虚拟环境

   ```
   conda activate envName
   或者
   activate envName
   或者
   source activate envName
   ```

4. 安装所需环境

   ```cmd
   pip install matplotlib
   pip install pyqt5
   conda install py-opencv
   ```
